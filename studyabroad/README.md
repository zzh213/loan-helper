# 出国读研选校匹配（DIY 助手）

输入**本科院校**（自动识别 985/211/双一流/双非）、**本科专业**（自动推荐匹配的研究生方向）、
均分或 GPA、雅思/GRE，立刻匹配「冲刺 / 匹配 / 保底」三档学校与项目，展示每个项目的硬性要求、
学费、时长、申请时间线；可注册登录、收藏选校清单，并一键生成**申请时间线 + 文书材料清单**。
目标：打破留学信息差。

## 运行

### 全栈（FastAPI 后端，推荐）

```bash
cd studyabroad
./run.sh            # 或 PORT=8200 ./run.sh
# 浏览器打开 http://127.0.0.1:8200
```

依赖：`fastapi`、`uvicorn`、`pydantic`、`pypdf`、`python-multipart`、`openpyxl`、`reportlab`
（见 `studyabroad/requirements.txt`，可 `python3 -m pip install -r studyabroad/requirements.txt`；账号/收藏仅用 Python 标准库）。

### 纯静态（零依赖，无账号、收藏存本地）

```bash
cd studyabroad && python3 -m http.server 8123
```

## 功能

- **中国高校录入**：224 所（39 985 / 75 211 / 31 双一流 / 76 双非 / 9 中外合作大学），搜索选择并自动识别层次
- **🌏 中外合作大学专项处理**：西交利物浦 / 宁波诺丁汉 / 上纽 / 昆山杜克 / 温州肯恩 / 北师港浸大 / 港中深 / 深北莫 / 广以理 —
  自动识别其申研时以**合作方海外学位**申请（英国荣誉学位 / 美国 4.0 GPA / 香港体系），按【海本/中外合作】层次评估，
  全英文授课背景给出**语言豁免提示**（雅思未达标不直接降档），并标注成绩体系换算口径，结果更全面客观
- **本科专业 → 研究生方向推荐（AI 智能识别 + 多选）**：填本科专业后，按相关度推荐研究生方向；
  即使填了列表外的专业（如「智能科学与技术」「电子商务」），也会用关键词智能识别给出相关研究生方向；
  研究生方向支持**多选**（可同时申请多个方向，结果取并集）且**可自定义输入**
- **GPA 换算**：百分制 / 4.0 / 4.3 / 5.0 自动估算
- **WES iGPA 估算 + 认证说明**：按 WES 段位法把百分制估算为美国 4.0 iGPA（A/B/C），并内置 WES 认证科普（谁需要、流程、费用 ~$205、耗时 4–8 周）；在**美国 / 加拿大**项目卡片上标注「通常需 WES/NACES 学历认证」
- **📊 申研参考案例库**：基于公开录取数据与往年规律整理的**对标参考画像**（明确标注为参考画像、非特定真人结果），
  21 条覆盖本科层次（985/211/双非/中外合办）× 专业方向（金融/CS/经济/商分/传媒/法律/工程/会计/管理/公共政策等），每条含背景 + 各校录取难度（可录取/可冲刺/偏高风险）+ 定位建议，帮你建立合理预期
- **📚 政策知识库**：11 大类 34 条深度政策科普，覆盖学位认证（WES/中留服/中外合办海外学位）、语言豁免、
  双非策略、rolling 递交时间、选校梯度、转专业、GRE/GMAT、各地区体系差异、签证（CAS/I-20）、就业身份（PSW/OPT/H-1B）、作品集/面试、中介与保录陷阱、费用奖学金等信息差痛点，支持分类 + 关键词搜索
- **三档匹配**：按「均分 − 该层次门槛」分为 🟢保底 / 🔵匹配 / 🟡冲刺，并提示语言/GRE 短板
- **具体到「每校每专业」**：项目名为真实硕士项目（如 LSE `Finance and Economics MSc`、华威 `Finance and Economics MSc`、格拉斯哥 `Economics, Banking & Finance MSc`），
  同一所学校录入多个专业方向（如 LSE 含金融 / 经济 / 会计金融 / 管理等 7 个），共 736 个项目覆盖 70+ 所院校（英 / 美 / 港 / 新 / 澳 / 加 / 爱尔兰 7 个国家地区，含加拿大多伦多/UBC/麦吉尔/滑铁卢、爱尔兰圣三一/UCD），重点铺开金融 / 经济 / 商科 / CS / 数据科学 / 传媒 / 法律集群
- **硬性要求 + 学费 / 时长 / 时间线**：每个项目一目了然
- **用户账号体系**：注册 / 登录（密码 PBKDF2 加盐哈希），收藏清单按账号隔离、跨设备保存；游客收藏存本地，登录后自动并入账号
- **选校清单**：⭐ 一键收藏
- **申请进度看板**：每个收藏项目可设置申请状态（待定/准备中/已递交/面试中/已录取/已拒绝）、截止日期与备注；
  顶部按状态统计、临近截止（≤14 天）自动提醒；登录后跨设备保存，游客存本地
- **申请时间线 + 文书清单**：基于收藏项目自动生成材料清单（成绩单/CV/PS/推荐信/语言成绩；含 GRE/作品集/语言测试等按需项）与分阶段申请时间线、各地区提示
- **软实力背景评分**：可逐条填写 **实习**（公司 / 岗位 / 时间 / 内容，自动识别名企加成）、**项目经历**（名称 / 角色 / 技术栈）、
  **校园活动**（社团 / 职务）、**交换 / 海外经历**（学校 / 形式），以及论文（SCI-EI/核心/会议/在投）、
  竞赛（国家级-省级-校级）、科研、工作经验、**证书 / 资格**（每行一项），系统计算竞争力等级并给予最多等效 **+6 分均分加成**计入匹配
- **🤖 简历 PDF 智能解析**：上传 PDF 简历，自动提取院校 / 专业 / 均分 / GPA / 雅思托福 / GRE / 实习（含公司岗位）/ 论文 / 竞赛 / 科研 / 证书，一键回填表单
- **✍️ AI 文书辅助**：基于背景与选校生成 PS/SOP 结构提纲、CV 撰写建议与竞争力点评（提供写作指引，不代写）
- **⬇️ 导出**：选校清单 + 申请进度一键导出 **Excel**；完整申请规划（清单 + 文书清单 + 时间线）导出 **PDF**
- **🔗 数据溯源 + 核实徽章**：每个项目标注「✔ 官方核实（含核实年月，跳官方项目页）」或「≈ 参考估算」，让用户清楚分辨数据可信度
- **📥 真实数据接入管道**：`data/sources/*.csv` 填入官方页核实数据 → `python3 scripts/ingest.py` 导入，自动升级为官方核实并写入官方 URL（详见 `data/sources/README.md`）

## 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/universities` | 全国高校 |
| GET | `/api/majors` | 本科专业 + 推荐研究生方向 |
| POST | `/api/recommend-fields` | 按本科专业智能识别推荐研究生方向（支持列表外专业） |
| GET | `/api/programs` | 项目列表 |
| POST | `/api/match` | 提交背景返回三档匹配 |
| POST | `/api/register` / `/api/login` / `/api/logout` | 账号 |
| GET/POST/DELETE | `/api/shortlist` | 收藏清单（需登录，按用户隔离） |
| PATCH | `/api/shortlist` | 更新申请进度（状态/截止日/备注） |
| GET | `/api/checklist` | 按收藏生成时间线+文书清单 |
| POST | `/api/checklist` | 游客模式：传 program_ids 生成清单 |
| POST | `/api/extract-resume` | 上传 PDF 简历，返回提取的结构化背景 |
| POST | `/api/sop` | 生成 PS 提纲 + CV 建议 + 竞争力点评 |
| POST | `/api/export/shortlist.xlsx` | 导出选校清单 + 进度为 Excel |
| POST | `/api/export/plan.pdf` | 导出完整申请规划为 PDF |

> 简历解析依赖 `pypdf`、`python-multipart`；导出依赖 `openpyxl`、`reportlab`（PDF 用 STSong-Light 内置中文字体）。

收藏 / 清单接口通过请求头 `Authorization: Bearer <token>` 识别用户。

## 分档逻辑（match.js / app.py 一致）

按「用户均分 + 软背景加成 − 该层次门槛」分档：≥+3 保底；0~+3 匹配；-3~0 冲刺；<-3 超出。
软背景加成最多等效 +6 分（论文/竞赛/实习/科研/工作经验加权，各有上限）；语言 / GRE 未达标会下调档位并提示。
