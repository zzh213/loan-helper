📄 什么是 WES 认证？申美/加是否需要？

WES（World Education Services） 是北美最常用的第三方学历学位认证机构（NACES 成员，同类还有 ECE / SpanTran）。它把你的中国成绩单按课程逐门换算成美国 4.0 制 GPA（iGPA），并出具官方认证报告。

  • 谁需要：主要是美国 / 加拿大部分院校的研究生申请要求 WES/NACES 认证；英国 / 港新 / 欧洲一般不需要（用学位证 + CDGDC 翻译认证即可）。
  • 流程：WES 注册建档 → 国内通过学信网 / 学位网（CDGDC）做电子认证并直接发送给 WES → WES 评估出报告。
  • 费用：Course-by-course（逐门课）约 $205，另加学信网认证费（约 ¥360/份）。
  • 耗时：WES 收到材料后约 7 个工作日；含学信网认证 + 传输，整体通常 4–8 周，建议尽早办理。

⚠️ 下方 GPA 换算为估算，仅供匹配参考。WES 实际按各门课程学分加权逐门换算，最终 iGPA 以官方报告为准。
